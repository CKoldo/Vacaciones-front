
  # Modulo de registro de vacaciones

  This is a code bundle for Modulo de registro de vacaciones. The original project is available at https://www.figma.com/design/VyRH2OBF8zRJlHx1GRSNXz/Modulo-de-registro-de-vacaciones.

  ## Modulos principales

  - **Registro de Personal:** administra los datos maestros de colaboradores y sus fechas de ingreso para calcular periodos vacacionales.
  - **Cronograma de Vacaciones:** planifica los rangos de días en bloque o flexibles que cada colaborador tomará durante el año.
  - **Visualización de Vacaciones:** centraliza el resumen de días tomados y pendientes, resaltando alertas sobre límites reglamentarios.
  - **Reprogramaciones:** permite mover rangos próximos a vencer, generando nuevos periodos y conservando el historial asociado.
  - **Administración:** gestiona los identificadores ESINAD y otros metadatos de soporte para los rangos de vacaciones registrados.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
